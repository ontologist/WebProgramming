What You'll Learn

Starting from zero programming experience, you will learn web development by building a complete Breakout game.

  • Phase 1 - HTML & CSS Foundations (Sessions 1-3): Build a personal website using semantic HTML5 and CSS3 styling
  • Phase 2 - Procedural JavaScript (Sessions 4-9): Learn programming fundamentals by building a Breakout game from an empty canvas
  • Phase 3 - Object-Oriented JavaScript (Sessions 10-13): Refactor the game using classes, encapsulation, and modularization
  • Phase 4 - Presentation & Review (Sessions 14-15): Showcase your game and reflect on skills learned

🎮 Game tutorial inspired by: MDN 2D Breakout Game using Pure JavaScript
