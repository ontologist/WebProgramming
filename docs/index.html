What You'll Learn / 学習内容

English: Students will learn about syntax and semantics of programming languages. They will understand data types and type checking, data scope, binding, and environments. Students will learn about function types and logic programming paradigms, and how to compare with other paradigms. In this course, students will acquire knowledge and hands-on experience about the computer language paradigm through building a complete Breakout game.

日本語: プログラミング言語の構文およびセマンティクスを学びます。データタイプとタイプチェック、スコープ、バインディング、および環境を理解します。関数型とロジックプログラミングパラダイム、および他のパラダイムとの比較のやり方について学習し、ブロック崩しゲームの構築を通じて実践的な経験を積みます。

  • Phase 1 - HTML & CSS Foundations: HTML5 semantic elements, CSS3 styling, Flexbox layout, personal website construction
  • Phase 2 - JavaScript & Canvas: Variables, data types, Canvas API, game loop, collision detection, user input handling
  • Phase 3 - Object-Oriented Programming: Classes, constructors, encapsulation, modularization, Audio API, localStorage
  • Phase 4 - Final Project: Game presentations, code review, web development skills reflection
