What is CSS?

CSSとは？

Cascading Style Sheets

CSS controls the LOOK of your web page — colors, fonts, spacing, layout.

Think of it this way:

  • HTML = Skeleton — the bones and structure of a body
  • CSS = Skin & Clothes — the appearance, colors, and style

Key Principle: CSS separates content (HTML) from presentation (CSS). This means you can change how a page looks without changing the HTML.

カスケーディングスタイルシート

CSSはウェブページの見た目を制御します — 色、フォント、余白、レイアウト。

こう考えてください：

  • HTML = 骨格 — 体の骨と構造
  • CSS = 肌と服 — 外見、色、スタイル

重要な原則：CSSはコンテンツ（HTML）とプレゼンテーション（CSS）を分離します。HTMLを変えずに見た目を変えられます。

3 Ways to Add CSS

CSSを追加する3つの方法

1. Inline CSS

<p style="color: red; font-size: 18px;">Hello!</p>

Applies to a single element only. Hard to maintain.

2. Internal CSS

<head> <style> p { color: red; font-size: 18px; } </style> </head>

Applies to the entire page, but only that one page.

3. External CSS (Best Practice)

<head> <link rel="stylesheet" href="style.css"> </head>

Why external is best: One CSS file can style ALL your pages. Change once, update everywhere!

1. インラインCSS

単一の要素にのみ適用。保守が困難。

2. 内部CSS

ページ全体に適用、ただしそのページだけ。

3. 外部CSS（ベストプラクティス）

外部が最良の理由：1つのCSSファイルで全ページをスタイリング。一度変更すればすべて更新！

CSS Syntax

CSS構文

The Basic Pattern

selector { property: value; }
  • Selector — targets which HTML element(s) to style
  • Property — what aspect to change (color, size, spacing...)
  • Value — how to change it (blue, 16px, center...)

Example

p { color: blue; font-size: 16px; }

This makes every paragraph on the page blue with 16px text.

基本パターン

  • セレクタ — どのHTML要素をスタイルするか指定
  • プロパティ — 何を変えるか（色、サイズ、余白...）
  • 値 — どう変えるか（blue, 16px, center...）

上の例はページ上のすべての段落を青色・16pxテキストにします。

Selectors: Element, Class, ID

セレクタ：要素、クラス、ID

Element Selector

p { color: green; }

Targets all <p> elements.

Class Selector

.highlight { background-color: yellow; }

Targets elements with class="highlight". Reusable on many elements.

ID Selector

#header { background-color: navy; color: white; }

Targets the one element with id="header". Unique per page.

When to use each:

  • Element — broad styling for all of that type
  • Class (.) — most common, reusable styles
  • ID (#) — one unique element only

使い分け

  • 要素セレクタ — そのタイプの全要素に広くスタイリング
  • クラス (.) — 最も一般的、再利用可能なスタイル
  • ID (#) — ユニークな1つの要素のみ

クラスセレクタが最もよく使われます。

色とテキスト

Common Text & Color Properties

  • Named: red, blue, green
  • Hex: #ff0000, #2563eb
  • RGB: rgb(255, 0, 0)

Text Properties

  • font-family: typeface
  • font-size: px, em, rem
  • font-weight: bold, normal, 700
  • text-align: left, center, right

よく使うテキスト・色プロパティ

  • color: テキストの色
  • background-color: 背景色
  • font-family: フォント（書体）
  • font-size: 文字サイズ（px, em, rem）
  • font-weight: 太さ（bold, normal）
  • text-align: 配置（left, center, right）

The Box Model

ボックスモデル

Every Element is a Box

In CSS, every HTML element is treated as a rectangular box with four layers:

Margin (outside spacing)
  ↓
Border (visible edge)
  ↓
Padding (inside spacing)
  ↓
Content (text, image, etc.)

Example

div { margin: 20px; /* space outside the box */ border: 2px solid black; padding: 15px; /* space inside the box */ width: 300px; /* content width */ }

すべての要素はボックスです

CSSでは、すべてのHTML要素が4つの層を持つ長方形のボックスとして扱われます。

  • Margin（マージン） — ボックスの外側の余白
  • Border（ボーダー） — 目に見える境界線
  • Padding（パディング） — ボックスの内側の余白
  • Content（コンテンツ） — テキストや画像など

Margins & Padding

マージンとパディング

Margin (Outside)

Controls spacing between elements.

/* All four sides */ margin: 20px; /* Top/Bottom Left/Right */ margin: 10px 20px; /* Individual sides */ margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px;

Padding (Inside)

Controls spacing inside the element, between content and border.

/* All four sides */ padding: 15px; /* Top/Bottom Left/Right */ padding: 10px 30px; /* Individual sides */ padding-top: 10px; padding-right: 30px; padding-bottom: 10px; padding-left: 30px;

Quick Rule: Margin = space outside the box. Padding = space inside the box.

簡単なルール

  • Margin = ボックスの外側のスペース（要素間の距離）
  • Padding = ボックスの内側のスペース（コンテンツと境界線の間）

ショートハンド記法で上下左右をまとめて指定できます。

Introduction to Flexbox

Flexboxの紹介

The Easiest Way to Layout Elements

Flexbox makes it simple to arrange items in a row or column, with automatic spacing and alignment.

/* Make a container a flex container */ .nav-bar { display: flex; justify-content: space-between; align-items: center; } /* HTML */ <nav class="nav-bar"> <a href="index.html">Home</a> <a href="about.html">About</a> <a href="hobbies.html">Hobbies</a> </nav>

Key Flexbox Properties:

  • display: flex; — activates Flexbox on the container
  • justify-content: — horizontal alignment (flex-start, center, space-between, space-around)
  • align-items: — vertical alignment (flex-start, center, stretch)

最も簡単なレイアウト方法

Flexboxは要素を行や列に配置し、自動的に間隔と配置を調整します。

  • display: flex; — コンテナでFlexboxを有効化
  • justify-content: — 水平方向の配置
  • align-items: — 垂直方向の配置

ナビゲーションバーの作成に最適です。

External Stylesheets

外部スタイルシート

Step 1: Create style.css

Create a new file called style.css in the same folder as your HTML files.

/* style.css */ body { font-family: Arial, sans-serif; margin: 0; padding: 0; background-color: #f5f5f5; } header { background-color: #2563eb; color: white; padding: 20px; text-align: center; }

Step 2: Link to Every HTML Page

<head> <meta charset="UTF-8"> <title>My Site</title> <link rel="stylesheet" href="style.css"> </head>

Result: One CSS file styles index.html, about.html, hobbies.html — all pages look consistent!

ステップ1：style.cssを作成

HTMLファイルと同じフォルダにstyle.cssという新しいファイルを作成します。

ステップ2：すべてのHTMLページにリンク

<link>タグを各ページの<head>内に追加します。

結果：1つのCSSファイルでindex.html, about.html, hobbies.html全てに統一感のあるスタイルを適用！

Styling MySite

MySiteのスタイリング

Practical Example: MySite style.css

/* === Header === */ header { background-color: #2563eb; color: white; padding: 20px; text-align: center; } /* === Navigation === */ nav { display: flex; justify-content: center; gap: 20px; background-color: #1e40af; padding: 10px; } nav a { color: white; text-decoration: none; font-weight: bold; } /* === Main Content === */ main { max-width: 800px; margin: 20px auto; padding: 20px; } /* === Footer === */ footer { background-color: #333; color: white; text-align: center; padding: 15px; }

Before: Plain white page, default fonts, no spacing control.

After: Colored header, organized nav bar, centered content, styled footer!

実践例：MySiteのstyle.css

適用前：白いページ、デフォルトフォント、余白の制御なし。

適用後：カラフルなヘッダー、整理されたナビバー、中央配置のコンテンツ、スタイリッシュなフッター！

CSSを追加するだけでサイトの見た目が大きく変わります。
