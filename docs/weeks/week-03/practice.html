About This Practice / この練習について

This is a guided in-class practice exercise. We will work through it together during class. You will create an external stylesheet for your MySite template, practicing each CSS concept from today's lecture one step at a time.
これは授業内のガイド付き練習です。授業中に一緒に進めていきます。今日の講義で学んだ各CSSコンセプトを一つずつ実践しながら、MySiteテンプレート用の外部スタイルシートを作成します。

This exercise is not graded. Its purpose is to give you hands-on experience with CSS before you start Assignment 1. Every technique you practice here will directly help you complete the assignment.
この練習は採点されません。課題1を始める前にCSSの実践経験を積むことが目的です。ここで練習するすべてのテクニックは、課題の完成に直接役立ちます。

What you need: Your MySite template folder with the 4 HTML pages (aboutme.htm, hobbies.htm, job.htm, game.htm) open in VS Code.

必要なもの： 4つのHTMLページ（aboutme.htm, hobbies.htm, job.htm, game.htm）が入ったMySiteテンプレートフォルダをVS Codeで開いてください。

Step 1: Create style.css and Link It / style.cssの作成とリンク

First, we will create a new external stylesheet and connect it to all of our HTML pages. This separates our styling from our content — a best practice in web development.
まず、新しい外部スタイルシートを作成し、すべてのHTMLページに接続します。これによりスタイリングとコンテンツが分離されます — Web開発のベストプラクティスです。

1a. Create the file

In VS Code, create a new file called style.css in the same folder as your .htm files. Right-click in the file explorer panel and select "New File", then type style.css.
VS Codeで、.htmファイルと同じフォルダにstyle.cssという新しいファイルを作成します。ファイルエクスプローラーパネルで右クリックし、「New File」を選択して、style.cssと入力してください。

1b. Link the stylesheet to ALL 4 pages

Open each of the 4 .htm files and add the following line inside the <head> section:
4つの.htmファイルそれぞれを開き、<head>セクション内に以下の行を追加してください：

<link rel="stylesheet" href="style.css">

1c. Remove the internal styles

In each .htm file, find the <style>...</style> block and either delete it or comment it out. We will rebuild all the styles in the external file.
各.htmファイルで、<style>...</style>ブロックを見つけて削除するかコメントアウトしてください。すべてのスタイルを外部ファイルで再構築します。

Tip: Save and check in browser — your pages should now look completely unstyled (plain text). That is correct! We will add all the styles back in the external file.

ヒント： 保存してブラウザで確認してください — ページは完全にスタイルなし（プレーンテキスト）に見えるはずです。それで正しいです！外部ファイルですべてのスタイルを追加し直します。

Step 2: Reset and Base Styles / リセットとベーススタイル

Every stylesheet should start with a reset and base styles. This ensures consistent behavior across different browsers and sets sensible defaults for your entire site.
すべてのスタイルシートはリセットとベーススタイルから始めるべきです。これにより、異なるブラウザ間で一貫した動作が保証され、サイト全体に適切なデフォルトが設定されます。

Open your new style.css file and add the following code:
新しいstyle.cssファイルを開き、以下のコードを追加してください：

/* Reset - removes default browser spacing */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* Base styles for the entire page */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #f5f5f5;
}

What each line does:

* { ... } — The universal selector. Applies to ALL elements on the page. We use it to remove default margins and padding that browsers add automatically.
ユニバーサルセレクタ。ページ上のすべての要素に適用されます。ブラウザが自動的に追加するデフォルトのマージンとパディングを削除するために使用します。

box-sizing: border-box — Makes padding and border included in the element's total width. This prevents layout surprises.
パディングとボーダーを要素の合計幅に含めます。レイアウトの予期しない挙動を防ぎます。

font-family — Sets the default font for all text on the page.
ページ上のすべてのテキストのデフォルトフォントを設定します。

line-height: 1.6 — Adds spacing between lines of text for readability.
テキストの行間にスペースを追加して読みやすくします。

color: #333 — Sets the default text color (dark gray, easier on the eyes than pure black).
デフォルトのテキスト色を設定します（ダークグレー、純粋な黒より目に優しい）。

background-color: #f5f5f5 — Sets a light gray background for the entire page.
ページ全体に薄いグレーの背景色を設定します。

Step 3: Style the Header (Colors and Typography) / ヘッダーのスタイル（色とタイポグラフィ）

Now let us make the header look professional. We will practice using background colors, gradients, text color, alignment, padding, and font sizing.
次に、ヘッダーをプロフェッショナルな見た目にしましょう。背景色、グラデーション、テキスト色、配置、パディング、フォントサイズを練習します。

Add the following to your style.css:
以下をstyle.cssに追加してください：

/* Header styling */
header {
    background: linear-gradient(135deg, #2563eb, #7c3aed);
    color: white;
    text-align: center;
    padding: 40px 20px;
}

/* Descendant selector: h1 inside header */
header h1 {
    font-size: 2rem;
    margin-bottom: 10px;
}

Selectors used here: You just used an element selector (header) and a descendant selector (header h1). The descendant selector targets only <h1> elements that are inside a <header> element.

使用したセレクタ： 要素セレクタ（header）と子孫セレクタ（header h1）を使いました。子孫セレクタは、<header>要素の中にある<h1>要素のみを対象にします。

Save and refresh your browser. Your header should now have a blue-to-purple gradient background with white centered text.
保存してブラウザを更新してください。ヘッダーに青から紫のグラデーション背景と白い中央揃えテキストが表示されるはずです。

Step 4: Style the Navigation with Flexbox / Flexboxでナビゲーションをスタイル

This is the key Flexbox exercise. We will turn the navigation list into a horizontal bar using display: flex. This is one of the most useful layout techniques in modern CSS.
これが重要なFlexbox演習です。display: flexを使って、ナビゲーションリストを水平バーに変えます。これは現代のCSSで最も便利なレイアウトテクニックの一つです。

Add the following to your style.css:
以下をstyle.cssに追加してください：

/* Navigation bar */
nav {
    background-color: #1f2937;
    padding: 10px 20px;
}

/* Remove bullet points and make horizontal with Flexbox */
nav ul {
    list-style: none;
    display: flex;
    gap: 5px;
    padding: 0;
}

/* Style the links */
nav a {
    color: white;
    text-decoration: none;
    padding: 10px 20px;
    border-radius: 5px;
}

/* Hover effect - changes when mouse is over the link */
nav a:hover {
    background-color: #4b5563;
}

Key concepts:

display: flex — This is the magic line! It makes the <ul> a flex container, which automatically places its children (the <li> items) side by side in a row.
display: flex — これが魔法の行です！<ul>をフレックスコンテナにし、子要素（<li>項目）を自動的に横並びに配置します。

gap: 5px — Adds space between flex items without needing margins.
gap: 5px — マージンを使わずにフレックスアイテム間にスペースを追加します。

:hover — A pseudo-class that applies styles only when the user's mouse is over the element.
:hover — ユーザーのマウスが要素の上にあるときのみスタイルを適用する擬似クラスです。

Tip: Try changing gap, justify-content, and background-color to see what changes! For example, add justify-content: center; to the nav ul rule to center the links.

ヒント： gap、justify-content、background-colorを変更して何が変わるか試してみてください！例えば、nav ulルールにjustify-content: center;を追加するとリンクが中央に配置されます。

Step 5: Style the Main Content (Box Model) / メインコンテンツのスタイル（ボックスモデル）

This is the box model exercise. We will use margin, padding, borders, and border-radius to create a clean content area. Understanding the box model is essential for controlling layout and spacing in CSS.
これがボックスモデルの演習です。マージン、パディング、ボーダー、ボーダーラディウスを使って、きれいなコンテンツ領域を作成します。ボックスモデルの理解は、CSSでレイアウトとスペーシングを制御するために不可欠です。

Add the following to your style.css:
以下をstyle.cssに追加してください：

/* Main content section */
section {
    max-width: 800px;
    margin: 30px auto;
    padding: 30px;
    background-color: white;
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

Box model breakdown:

margin: 30px auto — The top and bottom margin is 30px. The left and right margin is auto, which centers the element horizontally on the page.
margin: 30px auto — 上下のマージンは30px。左右のマージンはautoで、要素をページ上で水平方向に中央揃えします。

padding: 30px — Adds 30px of breathing room inside the box, between the border and the content.
padding: 30px — ボックスの内側、ボーダーとコンテンツの間に30pxの余白を追加します。

border-radius: 12px — Rounds the corners of the box.
border-radius: 12px — ボックスの角を丸くします。

box-shadow — Adds a subtle shadow beneath the box, creating a "floating card" effect.
box-shadow — ボックスの下に微妙な影を追加し、「浮いているカード」効果を作成します。

Try this: Open DevTools (F12) and hover over your section element. You will see the box model with margin (orange), padding (green), and content (blue) highlighted! This visual tool helps you understand exactly how spacing works.

試してみよう： DevTools（F12）を開き、section要素にカーソルを合わせてください。マージン（オレンジ）、パディング（緑）、コンテンツ（青）がハイライトされたボックスモデルが表示されます！このビジュアルツールでスペーシングの仕組みを正確に理解できます。

Step 6: Style the Articles (Class Selectors) / 記事のスタイル（クラスセレクタ）

Now we introduce class selectors — the most powerful and commonly used type of CSS selector. Unlike element selectors, classes let you target specific elements without affecting all elements of the same type.
ここでクラスセレクタを紹介します — CSSセレクタの中で最も強力で一般的に使用されるタイプです。要素セレクタと違い、クラスを使うと同じタイプのすべての要素に影響を与えずに特定の要素を対象にできます。

First: Add the class to your HTML

Open each .htm file and find the <article> elements. Add class="card" to each one:
各.htmファイルを開き、<article>要素を見つけてください。それぞれにclass="card"を追加します：

<!-- Before -->
<article>
    <h2>Your Title</h2>
    <p>Your content...</p>
</article>

<!-- After -->
<article class="card">
    <h2>Your Title</h2>
    <p>Your content...</p>
</article>

Then: Style the class in CSS

Add the following to your style.css:
以下をstyle.cssに追加してください：

/* Card styling using class selector */
.card {
    background-color: #fff;
    padding: 20px;
    margin-bottom: 20px;
    border-left: 4px solid #2563eb;
    border-radius: 8px;
}

/* Descendant selectors: elements inside .card */
.card h2 {
    color: #2563eb;
    margin-bottom: 10px;
}

.card p {
    line-height: 1.8;
}

Selectors used here: You just used a class selector (.card) and descendant selectors (.card h2, .card p). Classes are the most powerful selector — you can reuse them on any element! Any element with class="card" will receive these styles.

使用したセレクタ： クラスセレクタ（.card）と子孫セレクタ（.card h2、.card p）を使いました。クラスは最も強力なセレクタです — どの要素にも再利用できます！class="card"を持つすべての要素がこれらのスタイルを受け取ります。

Step 7: Style the Footer / フッターのスタイル

Finally, let us style the footer to match the professional look of our header. The footer uses familiar properties you have already practiced.
最後に、ヘッダーのプロフェッショナルな見た目に合わせてフッターをスタイルしましょう。フッターには、すでに練習した馴染みのあるプロパティを使います。

Add the following to your style.css:
以下をstyle.cssに追加してください：

/* Footer styling */
footer {
    background-color: #1f2937;
    color: #d1d5db;
    text-align: center;
    padding: 20px;
    margin-top: 30px;
}

Tip: You now have a complete external stylesheet! Every concept from today's lecture is in your style.css. Use this as a starting point for Assignment 1 — customize colors, fonts, spacing, and add more advanced techniques from W3Schools.

ヒント： これで完全な外部スタイルシートが完成しました！今日の講義のすべてのコンセプトがstyle.cssに入っています。これを課題1の出発点として使い、色、フォント、スペーシングをカスタマイズし、W3Schoolsからさらに高度なテクニックを追加してください。
