What is JavaScript? / JavaScriptとは？

The Third Language of the Web:

Remember our building analogy?

  • HTML -- the skeleton (structure)
  • CSS -- the paint and decoration (style)
  • JavaScript -- the electricity and brain (behavior)

JavaScript makes your web pages come alive. It can move things, respond to clicks, do math, and draw graphics!

Today is your first time writing JavaScript! Do not worry -- we will start simple and build up step by step.

今日がJavaScriptを初めて書く日です！心配しないでください。シンプルなところから段階的に学んでいきます。

Webの3番目の言語：

建物のたとえを覚えていますか？

  • HTML -- 骨格（構造）
  • CSS -- ペンキと装飾（スタイル）
  • JavaScript -- 電気と脳（動作）

JavaScriptはWebページに命を吹き込みます。物を動かしたり、クリックに反応したり、計算したり、グラフィックスを描いたりできます！

The <script> Tag / <script>タグ

Where Does JavaScript Go?

JavaScript code goes inside the <script> tag. Place it just before the closing </body> tag:

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello!</h1> <script> // Your JavaScript code goes here // あなたのJavaScriptコードはここに書く </script> </body> </html>

Why at the bottom? The browser reads your page from top to bottom. If we put the script at the bottom, the HTML is already loaded when our code runs.

JavaScriptはどこに書く？

JavaScriptのコードは<script>タグの中に書きます。</body>の直前に置きましょう。

なぜ下に置く？ブラウザはページを上から下へ読みます。スクリプトを下に置けば、コードが実行される時にはHTMLがすでに読み込まれています。

Variables: let and const / 変数：letとconst

What is a Variable?

A variable is like a labeled box that stores a value. You give it a name and put something inside.

// Declare a variable with let (value CAN change later) let score = 0; let playerName = "Yuki"; // Declare a constant with const (value CANNOT change) const canvasWidth = 480; const canvasHeight = 320;

Rules:

  • let -- use when the value might change later
  • const -- use when the value should never change

Naming rules: Start with a letter, no spaces, use camelCase (like ballColor). Names are case-sensitive!

変数とは？

変数は値を保存するラベル付きの箱のようなものです。名前を付けて、中に何かを入れます。

  • let -- 後で値が変わる可能性がある場合に使う
  • const -- 値が絶対に変わらない場合に使う

命名ルール：文字で始める、スペースなし、キャメルケースを使う（例：ballColor）。大文字と小文字は区別されます！

Data Types / データ型

Two Basic Types for Today:

1. Numbers (数値):

let x = 100; // whole number (integer) let y = 3.14; // decimal number let sum = x + y; // math works! sum = 103.14

2. Strings (文字列) -- text inside quotes:

let color = "red"; let greeting = 'Hello, World!'; let name = "Canvas";

Numbers vs Strings:

42 is a number (you can do math with it)

"42" is a string (it is just text that looks like a number)

今日学ぶ2つの基本型：

  • 数値（Numbers） -- 計算に使える数字：100、3.14
  • 文字列（Strings） -- 引用符で囲まれたテキスト："red"、'Hello'

42は数値（計算できる）、"42"は文字列（数字に見えるただのテキスト）

The <canvas> Element / <canvas>要素

What is Canvas?

The <canvas> element is like a blank drawing board on your web page. You can use JavaScript to draw shapes, lines, text, and images on it.

<!-- In your HTML body: --> <canvas id="myCanvas" width="480" height="320"></canvas>

Important attributes:

  • id -- a unique name so JavaScript can find it
  • width -- how wide the canvas is (in pixels)
  • height -- how tall the canvas is (in pixels)

Think of it like this: The <canvas> tag creates an empty painting canvas. JavaScript is your paintbrush.

Canvasとは？

<canvas>要素は、Webページ上の白い画用紙のようなものです。JavaScriptを使って図形、線、テキスト、画像を描くことができます。

こう考えましょう：<canvas>タグは空のキャンバスを作ります。JavaScriptがあなたの絵筆です。

Getting the Drawing Context / 描画コンテキストの取得

Before You Can Draw:

You need to get the 2D rendering context -- this is the object that has all the drawing methods.

<script> // Step 1: Find the canvas element const canvas = document.getElementById("myCanvas"); // Step 2: Get the 2D drawing context const ctx = canvas.getContext("2d"); // Now you can use ctx to draw! </script>

What just happened?

  • document.getElementById("myCanvas") -- finds the canvas in your HTML
  • .getContext("2d") -- gets the toolbox for 2D drawing
  • ctx -- short for "context"; this is your drawing tool

描画する前に：

2D描画コンテキストを取得する必要があります。これがすべての描画メソッドを持つオブジェクトです。

  • document.getElementById("myCanvas") -- HTMLからcanvasを見つける
  • .getContext("2d") -- 2D描画のツールボックスを取得する
  • ctx -- "context"の略；これがあなたの描画ツール

The Coordinate System / 座標系

How Canvas Coordinates Work:

The canvas uses an (x, y) coordinate system, but it is different from math class:

  • (0, 0) is at the top-left corner
  • x increases going to the right
  • y increases going DOWN (not up!)
(0,0) -----> x increases (right) | | | v y increases (down) Example: (200, 150) means 200 pixels from left, 150 pixels from top

Warning: In math, y goes UP. In canvas, y goes DOWN. This is the most confusing part for beginners!

注意：数学ではyは上に向かいますが、canvasではyは下に向かいます。これが初心者にとって最も混乱するポイントです！

Canvas座標の仕組み：

  • (0, 0)は左上の角にある
  • xは右に行くほど増える
  • yは下に行くほど増える（上ではない！）

Drawing Rectangles / 長方形の描画

Three Rectangle Methods:

// Fill a solid rectangle ctx.fillRect(x, y, width, height); // Draw just the outline ctx.strokeRect(x, y, width, height); // Erase a rectangle area ctx.clearRect(x, y, width, height);

Example -- Draw a blue paddle:

// Set the color first ctx.fillStyle = "blue"; // Draw a rectangle at position (190, 280), size 100x20 ctx.fillRect(190, 280, 100, 20);
  • x, y = the top-left corner of the rectangle
  • width, height = how wide and tall it is

3つの長方形メソッド：

  • fillRect(x, y, width, height) -- 塗りつぶした長方形
  • strokeRect(x, y, width, height) -- 枠線だけの長方形
  • clearRect(x, y, width, height) -- 長方形の領域を消去

x, y = 長方形の左上の角の位置。width, height = 幅と高さ。

Drawing Circles / 円の描画

Using arc() to Draw Circles:

There is no "circle" method. Instead, we draw an arc that goes all the way around (360 degrees):

// Start a new shape ctx.beginPath(); // Draw a circle: arc(centerX, centerY, radius, startAngle, endAngle) ctx.arc(240, 160, 10, 0, Math.PI * 2); // Fill it with color ctx.fillStyle = "red"; ctx.fill(); // Close the path ctx.closePath();

Understanding the Parameters:

  • centerX, centerY = the center point of the circle
  • radius = how big the circle is
  • 0 = start angle (beginning of circle)
  • Math.PI * 2 = end angle (full circle = 360 degrees)

arc()を使って円を描く：

"circle"メソッドはありません。代わりに、一周する弧（360度）を描きます。

  • centerX, centerY = 円の中心点
  • radius = 円の大きさ（半径）
  • Math.PI * 2 = 完全な円（360度）

The Path Pattern / パスのパターン

Always Follow This Pattern for Circles:

When drawing shapes like circles, you must follow these steps in order:

// 1. Begin a new path (start fresh) ctx.beginPath(); // 2. Define the shape (arc, lineTo, etc.) ctx.arc(100, 100, 30, 0, Math.PI * 2); // 3. Set color and fill (or stroke for outline) ctx.fillStyle = "green"; ctx.fill(); // 4. Close the path ctx.closePath();

Think of it like a pen:

1. Pick up the pen (beginPath)

2. Draw the shape (arc)

3. Fill it with color (fill)

4. Put the pen down (closePath)

円を描くときは常にこのパターンに従う：

ペンのように考えましょう：

  1. ペンを持ち上げる（beginPath）
  2. 形を描く（arc）
  3. 色で塗りつぶす（fill）
  4. ペンを置く（closePath）

Colors and Styles / 色とスタイル

Setting Colors:

You must set the color before drawing the shape:

// Fill color (inside of shape) ctx.fillStyle = "red"; ctx.fillStyle = "#0095DD"; // hex color ctx.fillStyle = "rgb(0,149,221)"; // RGB color // Outline color ctx.strokeStyle = "black"; // Outline thickness ctx.lineWidth = 3;

Common Color Names:

  • "red", "blue", "green", "yellow", "orange", "purple", "white", "black"

Important: Set fillStyle BEFORE calling fillRect() or fill(). The color applies to everything drawn after it until you change it again.

色の設定：

図形を描く前に色を設定する必要があります。

  • fillStyle -- 塗りつぶしの色
  • strokeStyle -- 枠線の色
  • lineWidth -- 枠線の太さ

重要：fillRect()やfill()を呼ぶ前にfillStyleを設定しましょう。

Using Variables for Drawing / 変数を使った描画

Why Use Variables?

Instead of "magic numbers", store values in variables with meaningful names. This makes your code easy to read and change.

// Ball properties const ballX = 240; const ballY = 160; const ballRadius = 10; const ballColor = "#0095DD"; // Paddle properties const paddleWidth = 100; const paddleHeight = 20; const paddleX = 190; const paddleY = 280; const paddleColor = "green"; // Draw the ball ctx.beginPath(); ctx.arc(ballX, ballY, ballRadius, 0, Math.PI * 2); ctx.fillStyle = ballColor; ctx.fill(); ctx.closePath(); // Draw the paddle ctx.fillStyle = paddleColor; ctx.fillRect(paddleX, paddleY, paddleWidth, paddleHeight);

なぜ変数を使う？

「マジックナンバー」の代わりに、意味のある名前の変数に値を保存します。コードが読みやすく、変更しやすくなります。

Putting It All Together / すべてをまとめる

Complete Working Example:

<!DOCTYPE html> <html> <head> <title>My Canvas Drawing</title> </head> <body> <canvas id="myCanvas" width="480" height="320" style="border: 1px solid black;"></canvas> <script> const canvas = document.getElementById("myCanvas"); const ctx = canvas.getContext("2d"); // Ball const ballX = 240; const ballY = 160; const ballRadius = 12; ctx.beginPath(); ctx.arc(ballX, ballY, ballRadius, 0, Math.PI * 2); ctx.fillStyle = "#0095DD"; ctx.fill(); ctx.closePath(); // Paddle const paddleX = 190; const paddleY = 290; ctx.fillStyle = "#2563eb"; ctx.fillRect(paddleX, paddleY, 100, 15); </script> </body> </html>

完全な動作例：

このコードをコピーしてHTMLファイルに貼り付け、ブラウザで開いてみましょう。青いボールと青いパドルが表示されます！

Key Takeaways / 重要なポイント

What We Learned Today:

  • JavaScript adds behavior to web pages (placed in <script> tags)
  • Variables store values: let (can change) and const (cannot change)
  • <canvas> is a drawing surface; get its context with getContext("2d")
  • Coordinates: (0,0) is top-left, x goes right, y goes DOWN
  • fillRect() draws rectangles; arc() draws circles
  • fillStyle sets the color BEFORE drawing
