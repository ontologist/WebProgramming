1. Assignment Overview / 課題概要

In this assignment, you will create your first JavaScript drawing using the HTML5 Canvas. You will draw a ball and a rectangle (paddle) on the canvas, using variables to store all positions, sizes, and colors.
この課題では、HTML5 Canvasを使って初めてのJavaScript描画を作成します。変数を使ってすべての位置、サイズ、色を保存しながら、canvas上にボールと長方形（パドル）を描きます。

This is the first step toward building our Breakout game. The ball and paddle you draw today will eventually move and interact!
これはブロック崩しゲームを作るための最初のステップです。今日描くボールとパドルは、最終的には動いて相互作用するようになります！

Goal: Demonstrate that you understand variables, data types, the canvas coordinate system, and basic drawing methods.

目標：変数、データ型、canvas座標系、基本的な描画メソッドを理解していることを示す。

2. What You Need To Do / やるべきこと

Follow these steps to complete the assignment:
以下のステップに従って課題を完成させてください：

Step 1: Create the HTML File / HTMLファイルを作成する

Create a new file called canvas-drawing.html. Start with the HTML template that includes a <canvas> element and a <script> tag. See the Starter Code in Section 5 below.
canvas-drawing.htmlという新しいファイルを作成します。<canvas>要素と<script>タグを含むHTMLテンプレートから始めましょう。下のセクション5のスターターコードを参照してください。

Step 2: Get the Canvas Context / Canvasコンテキストを取得する

Inside the <script> tag, use document.getElementById() to find your canvas and getContext("2d") to get the drawing context.
<script>タグ内で、document.getElementById()でcanvasを見つけ、getContext("2d")で描画コンテキストを取得します。

Step 3: Declare Ball Variables / ボールの変数を宣言する

Create variables for the ball's position (x, y), radius, and color. Use const since these values will not change in this assignment.
ボールの位置（x, y）、半径、色の変数を作成します。この課題では値が変わらないのでconstを使います。

const ballX = ...;       // pick your x position
const ballY = ...;       // pick your y position
const ballRadius = ...;  // pick a radius (e.g., 10-20)
const ballColor = "..."; // pick a color

Step 4: Declare Paddle Variables / パドルの変数を宣言する

Create variables for the paddle's position (x, y), width, height, and color.
パドルの位置（x, y）、幅、高さ、色の変数を作成します。

const paddleX = ...;      // pick your x position
const paddleY = ...;      // pick your y position
const paddleWidth = ...;  // pick a width (e.g., 80-120)
const paddleHeight = ...; // pick a height (e.g., 10-20)
const paddleColor = "..."; // pick a DIFFERENT color from the ball

Step 5: Draw the Ball / ボールを描く

Use beginPath(), arc(), fillStyle, fill(), and closePath() to draw the ball using your variables.
beginPath()、arc()、fillStyle、fill()、closePath()を使って、変数を使いボールを描きます。

Step 6: Draw the Paddle / パドルを描く

Use fillStyle and fillRect() to draw the paddle using your variables.
fillStyleとfillRect()を使って、変数を使いパドルを描きます。

Step 7: Add Comments / コメントを追加する

Add comments to explain what each section of your code does. This helps you and others understand your code.
コードの各セクションが何をしているかを説明するコメントを追加してください。

Bonus: Add Extra Creativity / ボーナス：創造性を追加する

Want extra points? Add more shapes! Draw a background, add extra circles, create a simple scene. Show off your creativity!
追加点が欲しいですか？もっと図形を追加しましょう！背景を描いたり、円を追加したり、簡単なシーンを作ったりして、創造性を発揮してください！

5. Starter Code / スターターコード

Use this template to get started. Fill in the parts marked with // TODO:
このテンプレートを使って始めましょう。// TODOとマークされた部分を記入してください：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Assignment 3: Canvas Drawing</title>
</head>
<body>
    <h1>My Canvas Drawing</h1>
    <canvas id="myCanvas" width="480" height="320"
        style="border: 2px solid #333;"></canvas>

    <script>
        // ============================================
        // SETUP: Get the canvas and context
        // ============================================
        const canvas = document.getElementById("myCanvas");
        const ctx = canvas.getContext("2d");

        // ============================================
        // BALL VARIABLES
        // TODO: Replace the ... with your own values
        // ============================================
        const ballX = ...;         // x position of ball center
        const ballY = ...;         // y position of ball center
        const ballRadius = ...;    // radius of the ball
        const ballColor = "...";   // color of the ball

        // ============================================
        // PADDLE VARIABLES
        // TODO: Replace the ... with your own values
        // ============================================
        const paddleX = ...;       // x position of paddle (top-left)
        const paddleY = ...;       // y position of paddle (top-left)
        const paddleWidth = ...;   // width of the paddle
        const paddleHeight = ...;  // height of the paddle
        const paddleColor = "..."; // color of the paddle

        // ============================================
        // DRAW THE BALL
        // TODO: Use arc() to draw a circle
        // ============================================
        ctx.beginPath();
        ctx.arc(ballX, ballY, ballRadius, 0, Math.PI * 2);
        ctx.fillStyle = ballColor;
        ctx.fill();
        ctx.closePath();

        // ============================================
        // DRAW THE PADDLE
        // TODO: Use fillRect() to draw a rectangle
        // ============================================
        ctx.fillStyle = paddleColor;
        ctx.fillRect(paddleX, paddleY, paddleWidth, paddleHeight);

    </script>
</body>
</html>

Tip: The starter code already has the drawing commands set up. You just need to replace the ... with actual numbers and color strings. Once you do that, you should see shapes on your canvas!

ヒント：スターターコードには描画コマンドが既に設定されています。...を実際の数値と色文字列に置き換えるだけです。それだけでcanvas上に図形が表示されます！

7. Tips for Success / 成功のためのヒント

  • Start with the starter code and replace values one at a time. Test after each change to make sure nothing breaks.
    スターターコードから始めて、一度に1つずつ値を置き換えましょう。変更のたびにテストして何も壊れていないことを確認してください。
  • Use the browser console (F12) to check for errors. If something is not showing up, the console will usually tell you why.
    ブラウザコンソール（F12）を使ってエラーを確認しましょう。何かが表示されない場合、通常コンソールが理由を教えてくれます。
  • Remember that the canvas coordinate system starts at (0,0) in the top-left corner. If your shapes are not where you expect them, double-check your x and y values.
    Canvas座標系は左上の角の(0,0)から始まることを覚えておきましょう。図形が予想した場所にない場合は、xとyの値を再確認してください。
  • Choose colors that contrast well with each other and with the white background so you can clearly see your shapes.
    お互いに、また白い背景と良いコントラストを持つ色を選んで、図形がはっきり見えるようにしましょう。
  • Be creative with positioning! Try placing the ball in different locations, or make the paddle a unique size. This is YOUR drawing.
    配置に創造性を発揮しましょう！ ボールを色々な場所に置いてみたり、パドルをユニークなサイズにしてみてください。これはあなたの描画です。

8. Common Mistakes to Avoid / よくある間違い

  • Forgetting beginPath() before arc() -- the ball will not draw correctly without it.
    arc()の前にbeginPath()を忘れる -- これなしではボールが正しく描画されません。
  • Using hard-coded numbers instead of variables -- ctx.arc(240, 160, 10, ...) should be ctx.arc(ballX, ballY, ballRadius, ...).
    変数の代わりにハードコードされた数値を使う -- ctx.arc(240, 160, 10, ...)ではなくctx.arc(ballX, ballY, ballRadius, ...)にする。
  • Setting fillStyle after fillRect() or fill() -- the color must be set BEFORE the drawing command.
    fillRect()やfill()の後にfillStyleを設定する -- 色は描画コマンドの前に設定する必要があります。
  • Putting the <script> tag in the <head> instead of before </body> -- the canvas element will not exist yet when the script runs.
    </body>の前ではなく<head>に<script>タグを置く -- スクリプト実行時にcanvas要素がまだ存在しません。
  • Misspelling the canvas id -- if the HTML says id="myCanvas" but the JavaScript says getElementById("mycanvas"), it will not work (case-sensitive!).
    canvasのidのスペルミス -- HTMLがid="myCanvas"でJavaScriptがgetElementById("mycanvas")だと動きません（大文字小文字を区別します！）。
  • Forgetting quotes around color strings -- const color = red; will cause an error. It must be const color = "red";.
    色文字列の引用符を忘れる -- const color = red;はエラーになります。const color = "red";にする必要があります。
  • Drawing the ball or paddle completely outside the canvas boundaries -- make sure x and y values are within your canvas width and height.
    ボールやパドルをcanvasの境界の完全に外に描く -- xとyの値がcanvasの幅と高さの範囲内にあることを確認してください。
