What is the Web? / ウェブとは？

How a Web Page Reaches You:

  1. You type a URL (like www.google.com) into your browser
  2. Your browser sends a request across the internet to a server
  3. The server sends back HTML (and CSS, JavaScript, images...)
  4. Your browser reads the HTML and draws the page on your screen

Client-Server Model / クライアント・サーバーモデル

Client = your browser (Chrome, Firefox, Safari). It asks for pages.

Server = a computer somewhere in the world. It sends pages back.

クライアント = あなたのブラウザ（Chrome、Firefox、Safari）。ページを要求します。

サーバー = 世界のどこかにあるコンピュータ。ページを送り返します。

ウェブページがあなたに届くまで：

  1. URLを入力する（www.google.com のように）
  2. ブラウザがリクエストを送る（インターネットを通じてサーバーへ）
  3. サーバーがHTMLを返す（CSS、JavaScript、画像なども）
  4. ブラウザがHTMLを読んで画面にページを表示する

What is HTML? / HTMLとは？

HTML = HyperText Markup Language

HTML = ハイパーテキスト・マークアップ・ランゲージ

HTML is the skeleton of every web page. It tells the browser what content to show: headings, paragraphs, images, links, and more.

Important: HTML is NOT a programming language!

HTML is a markup language. It describes the structure of content. It does not have logic, loops, or conditions -- that is what JavaScript is for (we will learn it later).

Think of it this way:

  • HTML = the bones and organs (structure)
  • CSS = the skin and clothes (appearance)
  • JavaScript = the muscles and brain (behavior)

HTMLとは？

HTMLはすべてのウェブページの骨格です。ブラウザに何のコンテンツを表示するか指示します：見出し、段落、画像、リンクなど。

重要：HTMLはプログラミング言語ではありません！

HTMLはマークアップ言語です。コンテンツの構造を記述します。

  • HTML = 骨と臓器（構造）
  • CSS = 皮膚と服（見た目）
  • JavaScript = 筋肉と脳（動作）

HTML Document Structure / HTML文書の構造

Every HTML page follows the same basic structure. Here is the minimal template:

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first web page.</p> </body> </html>

What Each Part Does:

  • <!DOCTYPE html> -- Tells the browser "this is an HTML5 page"
  • <html> -- The root element; everything goes inside this
  • <head> -- Invisible settings (page title, character encoding)
  • <body> -- All the visible content goes here

各部分の役割：

  • <!DOCTYPE html> -- 「これはHTML5ページです」とブラウザに伝える
  • <html> -- ルート要素；すべてがこの中に入る
  • <head> -- 見えない設定（ページタイトル、文字エンコーディング）
  • <body> -- すべての見えるコンテンツがここに入る

Basic HTML Tags: Text / 基本的なHTMLタグ：テキスト

Headings (h1 to h6):

Headings show titles and section names. h1 is the biggest, h6 is the smallest.

<h1>Main Title</h1> <h2>Section Title</h2> <h3>Subsection Title</h3>

Paragraphs:

Use <p> to create a block of text.

<p>This is a paragraph of text.</p> <p>This is another paragraph.</p>

Emphasis:

Use <strong> for bold and <em> for italic.

<p>This word is <strong>bold</strong> and this is <em>italic</em>.</p>

テキスト用のHTMLタグ：

  • h1~h6 -- 見出し（h1が最大、h6が最小）
  • p -- 段落（テキストのブロック）
  • strong -- 太字（重要なテキスト）
  • em -- 斜体（強調テキスト）

Basic HTML Tags: Lists & Links / リストとリンク

Unordered List (bullet points):

<ul> <li>Apples</li> <li>Bananas</li> <li>Cherries</li> </ul>

Ordered List (numbered):

<ol> <li>First step</li> <li>Second step</li> <li>Third step</li> </ol>

Links:

The <a> tag creates a clickable link. The href attribute specifies where to go.

<a href="https://www.google.com">Go to Google</a> <!-- Open in a new tab --> <a href="https://www.google.com" target="_blank">Open Google in new tab</a>

リストとリンクのタグ：

  • ul -- 順序なしリスト（箇条書き）
  • ol -- 順序ありリスト（番号付き）
  • li -- リスト項目
  • a href="..." -- リンク（クリックで別のページへ）
  • target="_blank" -- 新しいタブで開く

Basic HTML Tags: Images / 画像

The <img> tag displays an image. It does not have a closing tag!

<img src="photo.jpg" alt="A beautiful sunset">

Important Attributes:

  • src -- the file path or URL of the image
  • alt -- a text description (shown if the image fails to load; also helps accessibility)

Relative vs. Absolute Paths:

<!-- Relative path: the image is in the same folder --> <img src="photo.jpg" alt="My photo"> <!-- Relative path: the image is in a subfolder --> <img src="images/photo.jpg" alt="My photo"> <!-- Absolute path: full URL from the internet --> <img src="https://example.com/photo.jpg" alt="My photo">

画像タグ：

  • img -- 画像を表示する（閉じタグなし！）
  • src -- 画像のファイルパスまたはURL
  • alt -- テキスト説明（画像が読み込めない場合に表示される）

相対パス：同じフォルダにあるファイル（例："photo.jpg"）

絶対パス：インターネット上の完全なURL（例："https://..."）

Tools Setup / ツールのセットアップ

1. VS Code (Text Editor):

Download from: https://code.visualstudio.com

VS Code is a free text editor where you will write all your code.

2. Recommended Extension: Live Server

This extension automatically refreshes your browser when you save a file. Very useful!

  • Open VS Code
  • Click the Extensions icon on the left sidebar (or press Ctrl+Shift+X)
  • Search for "Live Server" by Ritwick Dey
  • Click Install

3. File Naming Rules:

  • Always use the .html extension for HTML files
  • Use lowercase letters: index.html (not Index.HTML)
  • No spaces in file names: my-page.html (not my page.html)

必要なツール：

  • VS Code -- 無料のテキストエディタ（code.visualstudio.com）
  • Live Server拡張機能 -- ファイル保存時にブラウザを自動更新
  • ファイル名は小文字で、スペースなし、拡張子は.html

Developer Tools / 開発者ツール

Every browser has built-in developer tools. They let you see and even change the HTML of any web page!

How to Open Developer Tools:

  • Press F12 on your keyboard, or
  • Right-click on any web page and select "Inspect"

The Elements Tab:

This tab shows you the HTML structure of the page. You can:

  • See every HTML tag on the page
  • Click on an element to highlight it
  • Try it: Double-click on any text in the Elements tab and change it! (Don't worry -- it only changes on your screen, not the real website.)

Try it now! Open a new tab, go to any website, press F12, and find the Elements tab.

今すぐ試しましょう！ 新しいタブを開き、任意のウェブサイトに行き、F12を押して、Elementsタブを見つけてください。

開発者ツールの使い方：

  • F12を押す、または右クリックして「検証」を選択
  • ElementsタブでHTMLの構造を確認できる
  • テキストをダブルクリックして変更してみましょう（自分の画面でのみ変更されます）

Key Takeaways / 重要なポイント

What We Learned Today:

  • HTML = Structure. It defines what content appears on a web page.
  • Tags come in pairs: most tags have an opening tag and a closing tag (e.g., <p> ... </p>)
  • Every page needs: <!DOCTYPE html>, <html>, <head>, and <body>
  • Basic tags: h1-h6, p, strong, em, ul, ol, li, a, img
  • Tools: VS Code (to write code) + Browser (to view results)
