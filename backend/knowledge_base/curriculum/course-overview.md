# WP-200 Web Programming - Course Overview

## Course Information
- **Code:** WP-200 (Course Number: 200)
- **Title:** Web Programming / ウェブ・プログラミング
- **University:** Kwansei Gakuin University, School of Policy Studies
- **Campus:** Kobe Sanda Campus
- **Term:** Spring 2026
- **Schedule:** Tuesday, Period 2
- **Instructor:** Tijerino Y.A.
- **Credits:** 2
- **Target:** 2nd year students
- **Language:** English
- **Format:** Face-to-face (with Zoom broadcast and recording)
- **Prerequisites:** None - no prior programming experience required

## Course Purpose
Students will learn web development from zero programming experience by building a complete Breakout game. The course covers HTML5, CSS3, and JavaScript, progressing from basic web pages through procedural programming to object-oriented JavaScript.

## AI Policy
AI use is required for this course. AI is used as a complementary tool under professor supervision. All AI use must comply with university policies.

## Assessment
100% Continuous Assessment:
- Attendance and discussion participation
- Individual assignments (9 assignments throughout the course)
- Online quizzes
- No final examination
- No group work - all assignments are individual

## Course Structure

### Phase 1: HTML & CSS Foundations (Sessions 1-3)
Build a personal website (MySite) using semantic HTML5 and CSS3.

### Phase 2: Procedural JavaScript (Sessions 4-9)
Learn programming fundamentals by building a Breakout game from an empty canvas.
Based on MDN's "2D Breakout Game using Pure JavaScript" tutorial.

### Phase 3: Object-Oriented JavaScript (Sessions 10-13)
Refactor the game using classes, encapsulation, and modularization.

### Phase 4: Presentation & Review (Sessions 14-15)
Final project showcase and course reflection.
